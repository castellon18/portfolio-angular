# Portfolio

Este proyecto se generó utilizando [Angular CLI](https://github.com/angular/angular-cli) version 19.2.11.

## Descripción
Proyecto Personal, donde hace una descripción de mis habilidades y desempeño en los proyectos realizados para los clientes

## Modo desarrollo

Para iniciar un servidor de desarrollo local, ejecute:

```bash
npm install
ng serve
```

Una vez que el servidor esté en funcionamiento, abra su navegador y navegue a `http://localhost:4200/`. La aplicación se recargará automáticamente al modificar cualquier archivo fuente.
